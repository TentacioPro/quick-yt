Guidelines / Mobile

Mobile Accessibility & A11y

Ensuring QuickYT Android remains usable for everyone. These guidelines detail focus management, touch targets, and contrast requirements crucial for a compliant and inclusive experience.

Focus Ring Treatments

Focus indicators must be highly visible. We use the signature Accent Amber to provide stark contrast against both Slate surfaces and primary content.

Focus Ring Color

#FFB300 (Accent Amber)

Ring Width & Offset

4px solid / 2px offset

Touch Targets

Strict adherence to 48x48px minimum hit areas.

48dp
48dp
Visual component size may be smaller (e.g., 24x24 icon), but the tappable area must extend to meet 48dp criteria.

Interactive elements must provide context beyond their visual label. Use `contentDescription` on Android to articulate state and purpose.

// Example: Play Button Toggle if (isPlaying) { button.contentDescription = "Pause current track" } else { button.contentDescription = "Play current track" }

account_tree Semantic Headings

Maintain a logical, unbroken heading hierarchy (H1 -> H2 -> H3). Do not skip levels for styling purposes.

  • H1 Screen Title (One per page)
  • H2 Major Section Heading
  • H3 Subsection or Card Title
